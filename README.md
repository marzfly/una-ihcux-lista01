2) Meus comandos favoritos

────୨ atalhos de teclado ৎ────

seta para cima - volta o comando anterior
tab - terminal autocompleta
ctrl c - interrompe comando

────୨ comandos ৎ────

cd | mudar de diretório
cd .. | voltar para o diretório anterior
cls | limpar tela do cmd
mkdir | criar diretório
java -version | versão do java
dotnet --version | versão do .NET
dotnet --info | informações do dotnet
dotnet new list | lista todos as possibilidades
dotnet new console -n App | cria uma aplicação na pasta "App"
git --version | versão do git
node -v | versão do node
echo pronto > arquivo.txt | criar arquivo de texto
dir | exibir os arquivos do diretório onde eu estou
systeminfo | informações sobre a máquina
